Congratulations, your certificate has been approved and was retrieved from the Certification Authority. As a final step, you will want to install it into your webbrowser, Mozilla Firefox (change browser).

Please follow these steps:

  1. Press the install button to begin

    • Your webbrowser will open a new window
  2. When an Internet Security dialog appears, choose Allow.
  3. If you have set a master password, enter it in the dialog that pops up.
  4. In the password dialog for the certificate backup, enter the password tobaronetwothree Note that this a temporary password, and that it has been copied to your clipboard as well (use Ctrl-V to paste).
  5. The Certificate Import Wizard will appear, press Next twice.
  6. In the password field, enter the password tobaronetwothree Note that this a temporary password, and that it has been copied to your clipboard as well (use Ctrl-V to paste).
    • It is advised toenable strong private key protection for increased security.
  7. Press Next a couple of times, and Finish.
  8. If you enabled strong private key protection, confirm the keystore access by pressing Ok.
